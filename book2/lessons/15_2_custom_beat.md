# CHAPTER 15-2 · 나만의 음악과 영상으로 노트 만들기

!!! note "이번 차시에 하는 일"
    - 내가 준비한 **음악 파일(MP3)**과 **동영상 파일(MP4)**을 내 작업 폴더에 넣습니다
    - 게임 플레이 화면 뒤편에서 동영상이 배경 음악과 함께 **전체 화면으로 자동 재생**되도록 AI에게 부탁합니다
    - 음악을 들으며 방향키를 누르면 박자가 저장되는 **'악보 제작기(채보 녹음) 모드'**를 만들어 봅니다

> ⏱️ 걸리는 시간: 약 30분 · 🧰 준비물: 15차시 제작 결과물, 내가 좋아하는 음악 파일(MP3), 신나는 동영상 파일(MP4)

---

## 15-2.1 왜 이걸 하나요?

리듬게임 오락실 기계들을 보면, 단순히 음악만 나오는 게 아니라 뒤쪽 배경에 화려한 가수들의 무대 영상이나 애니메이션(BGA, 배경 영상)이 끊임없이 흘러나와 눈을 즐겁게 해 줍니다. 또한, 기본 제공 곡 대신 내가 가장 좋아하는 트로트 노래나 댄스 가요를 배경으로 게임을 하고 싶어집니다.

이번 차시에서는 게임에 **내가 원하는 음악과 배경 영상**을 넣는 법을 배웁니다. 그리고 코드를 전혀 모르더라도 노래 박자에 맞춰 키보드를 툭툭 누르면 그 순간의 박자를 그대로 기억하여 게임의 악보(노트 패턴)로 자동 변환해 주는 **'나만의 악보 제작 기능'**을 AI에게 부탁해 봅니다. 내가 직접 노트를 설계하여 진정한 나만의 리듬게임을 만들어 완성하는 클라이맥스 단계입니다.

---

## 15-2.2 따라 하기

### 15-2.2.1 단계 ① 내 작업 폴더에 음악과 동영상 파일을 넣습니다

내가 가지고 있는 음악 파일 하나와 동영상 파일 하나를 준비합니다. 
(이름을 각각 **`music.mp3`** 와 **`bg.mp4`** 로 알기 쉽게 미리 바꿔 둡니다.)

1. 바탕화면의 **`rhythm-game`** 폴더를 엽니다.
2. 그 안에 있는 **`assets`** (자산 폴더, 음악이나 이미지를 모아두는 곳) 폴더로 들어갑니다.
3. 준비한 `music.mp3` 와 `bg.mp4` 파일을 마우스로 끌어다 **`assets`** 폴더 안에 넣어 둡니다.

---

### 15-2.2.2 단계 ② 내 음악과 동영상을 화면 배경에 띄워 달라고 부탁합니다

Claude Code 터미널 창을 열어 아래의 프롬프트를 복사해 붙여넣습니다.

!!! quote "🗣️ 이대로 복사해서 붙여넣습니다 (AI에게 하는 말)"
    ```
    우리 프로젝트의 assets 폴더에 music.mp3 음악 파일과 bg.mp4 동영상 파일을 넣었어.
    
    1. 게임 플레이를 시작하면 bg.mp4 동영상이 게임 화면 바로 뒤편 전체 배경으로 소리 없이 끊임없이 반복 재생(반복 재생 모드)되도록 해줘.
    2. 동영상이 게임판 레이아웃(노트나 판정선)을 가리지 않도록 z-index를 사용해 가장 뒤쪽 레이어로 보내줘.
    3. 기존의 기본곡 대신에 assets/music.mp3 음악이 연주되도록 교체해줘.
    ```

승인을 물어보면 엔터를 누르고 잠시 기다립니다. 브라우저 화면 배경에 동영상이 자동으로 흘러나오고 음악이 교체되면 성공입니다.

### 15-2.2.3 단계 ③ 내가 직접 박자를 녹음하는 '악보 제작기'를 요청합니다

이제 음악을 들으며 손가락으로 박자를 쳐서 나만의 노트를 제작할 수 있는 녹음 도구를 만들어 달라고 Claude Code에게 부탁해 봅니다.

!!! quote "🗣️ 이대로 복사해서 붙여넣습니다 (AI에게 하는 말)"
    ```
    내가 노래를 들으며 키보드 방향키(← ↓ ↑ →)를 누르면, 그 누른 시점(ms, 밀리초 단위)과 어떤 방향키를 눌렀는지가 실시간으로 기록되는 '채보 녹음 모드' 기능을 추가해줘.
    
    - 화면 구석에 [채보 녹음 시작/중지] 버튼을 작게 만들어줘.
    - 녹음 시작을 누르면 음악이 0초부터 재생되며 내가 방향키를 누를 때마다 기록이 시작되게 해줘.
    - 녹음 중지를 누르면 지금까지 기록된 노트 리스트 데이터들을 JSON 형식의 문자열로 만들어 화면 한쪽에 큼직한 텍스트 창(글상자)으로 띄워줘. 내가 마우스로 긁어서 복사할 수 있게.
    ```

엔터를 눌러 코드를 적용한 뒤 브라우저 화면으로 돌아옵니다. 화면 한구석에 [채보 녹음] 버튼이 새로 생겼는지 확인합니다.

---

### 15-2.2.4 단계 ④ 박자를 쳐서 나만의 악보 데이터를 만들어 봅니다

이제 직접 연주가가 되어 악보를 녹음해 보겠습니다.

1. 화면의 **[채보 녹음 시작]** 버튼을 누릅니다. 음악이 흘러나오기 시작합니다.
2. 음악 박자에 맞춰 키보드의 **화살표 방향키(← ↓ ↑ →)**를 리듬감 있게 툭툭 눌러 줍니다. 누르는 박자가 실시간으로 기록됩니다.
3. 노래가 끝나거나 원하는 구간까지 쳤다면 **[채보 녹음 중지]** 버튼을 누릅니다.
4. 화면에 영어와 숫자로 복잡하게 적힌 텍스트 상자(예: `[{"time": 1020, "lane": 0}, ... ]`)가 팝업처럼 큼직하게 뜹니다.
5. 이 텍스트 상자 안의 글씨를 마우스로 긁어서 전부 블록으로 지정한 뒤 **`Ctrl + C` (복사하기)** 단축키를 눌러 내 컴퓨터에 복사해 둡니다.

---

### 15-2.2.5 단계 ⑤ 복사한 악보를 내 게임에 적용해 달라고 부탁합니다

방금 복사한 텍스트 데이터를 그대로 Claude Code에게 던져서 게임에 주입해 달라고 말만 하면 완성됩니다.

!!! quote "🗣️ 이대로 복사해서 붙여넣습니다 (AI에게 하는 말)"
    ```
    방금 채보 녹음 기능으로 추출한 나만의 노트 데이터야.
    이 데이터들을 우리 게임의 NORMAL 난이도 노트 패턴으로 교체해줘:
    
    [여기에 복사해 둔 텍스트 데이터를 Ctrl+V로 붙여넣습니다]
    ```

적용이 완료되면, 난이도 선택에서 **NORMAL**을 누르고 게임을 시작해 확인합니다. 방금 전 내가 리듬에 맞춰 툭툭 눌렀던 박자 그대로 노트가 위에서 내려오는 신기한 경험을 할 수 있습니다!

---

!!! success "✅ 여기까지 됐으면"
    - ☐ 내 음악 파일(`music.mp3`)과 동영상 파일(`bg.mp4`)을 `assets` 폴더에 넣었다
    - ☐ 내 동영상이 게임 배경에서 반복 재생되고, 새 음악으로 교체된 것을 확인했다
    - ☐ [채보 녹음] 기능으로 직접 리듬 쳐서 나만의 악보(JSON 데이터)를 뽑아 복사했다
    - ☐ 복사한 데이터를 AI에게 주어 게임에 적용하고, 내가 만든 노트대로 게임이 실행됨을 확인했다

!!! abstract "📌 핵심 요약"
    - **배경 영상(BGA)**은 게임 화면 맨 뒤 레이어(z-index)에 동영상을 반복 재생하게 배치하여 오락실 느낌을 살리는 기술입니다.
    - **채보(악보) 녹음**은 음악의 타임라인에 맞춰 키를 누르는 시점(ms)을 배열에 순차적으로 담는 소프트웨어 로직입니다.
    - 프롬프트에 **"복사 가능한 텍스트 창"**으로 데이터를 뽑아달라고 명시하면, 독자가 번거롭게 소스 코드를 열지 않고도 악보 데이터를 직접 코드로 주입하는 다리 역할을 AI가 대신해 줍니다.

!!! info "🔎 낱말 사전"
    - **BGA (배경 영상)** — Background Animation의 준말. 리듬게임 플레이 뒤편에서 흘러나오는 비주얼 뮤직비디오나 그래픽 동영상 배경.
    - **채보 (악보 만들기)** — 음악을 듣고 노트가 떨어질 시점과 위치를 기록하여 게임용 데이터 파일로 만드는 작업.
    - **JSON** — 컴퓨터와 사람이 데이터를 주고받기 편하게 영어와 숫자 쌍(괄호 형태)으로 표기하는 텍스트 규칙 형식.

> **다음 차시 예고** — 16차시에서는 나만의 개성이 들어간 이 게임을 **내 취향에 맞는 색깔, 글꼴(폰트), 문구, 콤보 반짝임 효과** 등으로 화려하게 단장하여 디자인 완성도를 최고조로 올려 봅니다.
